# SiteCheck Demo

דמו החלטה מהיר ליזם לפי שכבת ArcGIS אחת של ייעודי קרקע.

## מה הדמו עושה

- קלט: מיקום (לחיצה על מפה או גוש/חלקה), סוג מיזם, שטח נדרש.
- טעינה בזמן אמת מ־ArcGIS (`ייעודי_קרקע`, שכבה 0).
- מיפוי ל־5 קטגוריות ייעוד בלבד: חקלאי / תיירות־נופש / מגורים / מסחרי / שטח פתוח־שמורה.
- החלטה ירוק/צהוב/אדום לפי מטריצה קשיחה + כלל שטח:
  - אם `שטח נדרש > שטח חלקה` => אדום.
  - אם `שטח נדרש > 70% משטח חלקה` ורמת בסיס הייתה ירוק => צהוב.

## הרצה

```bash
npm install
npm run dev
```

## שכבת ArcGIS שבשימוש

- Service: `https://services1.arcgis.com/yAQXemoDSgzdfV2A/arcgis/rest/services/ייעודי_קרקע/FeatureServer`
- Layer: `0` (שם השכבה: `הדר`)
- שדות מרכזיים:
  - `Ystr` - טקסט ייעוד
  - `Shape__Area` - שטח מגרש במ"ר
  - `Gush`, `Helka` - זיהוי חלקה

## מחקר שכבות להמשך (אם מרחיבים מעבר לדמו)

כרגע הדמו נשאר בכוונה עם שכבה אחת בלבד. אם תרצה גרסה "חצי-מוצר", אלה השכבות הבאות שכדאי לשלב:

1. גבולות חלקות/קדסטר עדכניים (לדיוק בחיתוך מיקום והצגת פוליגון).
2. שמורות טבע/גנים לאומיים/שטחים מוגנים (בדיקת חסמים רגולטוריים מוקדמים).
3. תב"עות מאושרות ותב"עות בהפקדה (מצב תכנוני וודאות סטטוטורית).
4. ייעודי דרך/תשתיות על (סינון חלקות עם מגבלות תשתית חזקות).

שילוב 2-4 לא נדרש לדמו הנוכחי, אבל ייתן קפיצה משמעותית באמינות העסקית.
